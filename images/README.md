## Tips
- Use JPG format for photographs
- Optimize images for web (compress them to reduce file size)
- Recommended dimensions: 800-1600px width for hero images, 600-800px for gallery items
- Keep file sizes under 500KB when possible for faster loading

The site includes placeholder images that will show if your images are not yet added.
